3D-Renderer

This simple 3D rendering engine, crafted in C, serves as my hands-on introduction to the basics of 3D rendering. It incorporates dynamic lighting, texture mapping, and loading .obj files. I used SDL2 for texture handling and pixel rendering.

GitHub repository

I developed this project to grasp the basics of 3D rendering, using only C and SDL2. The renderer showcases various techniques, including per-vertex and per-pixel lighting, flat shading, and the ability to load objects with or without textures. All features demonstrated in this demo.
