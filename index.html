Airbnb-Mockup

Service-Oriented Accommodation Booking Platform.
I led the development of this Airbnb-inspired platform utilizing Golang and a service-oriented architecture within a four-member team.

We orchestrated database infrastructure, integrating Cassandra, MongoDB, and Consul, alongside HDFS for distributed image storage and Redis for caching.

We used Docker and Docker Compose for containerization.
Delivered a robust and scalable solution for accommodation discovery and booking processes.

Project includes following services:

  • User authentication
  • Profile management
  • Accommodations
  • Reservations
  • Recommendations
  • Notifications
